Recipe API

Create Read Update and Delete recipes

Recipes

Route Method Description Example 
/recipes
 GET Gets a list of all recipes stored in the API List Recipes 
/recipes/1
 GET Returns a single single recipe with instructions and ingredients Get Single Recipe 
/recipes
 POST Adds a new recipe for the user 
curl -d '{ title: 'Avocado Toast', description: 'Open faced sandwich with mashed avocado over toast', prepTime: 2, cookTime: 3, totalTime: 5, servings: 1 }' -H 'Content-Type: application/json' ''
 
/recipes/1
 PUT Updates recipe information 
/recipes/1
 DELETE Deletes a recipe from the Database

Instructions

Route Method Description Example 
/instructions
 GET Gets a list of all instructions stored in the API List Instructions 
/instructions/1
 GET Returns a single instruction as well as the recipe it belongs to Get Single Instruction 
/instructions
 POST Adds a new instruction for the user 
curl -d '{ recipeId: 1, stepNumber: 4, description: 'Enjoy!' }' -H 'Content-Type: application/json' ''
 
/instructions/1
 PUT Updates instruction information 
/instructions/1
 DELETE Deletes an instruction from the Database
